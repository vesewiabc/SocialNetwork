# LinkA 🔗

LinkA — социальная сеть, созданная для простого и живого общения.  
Мы соединяем людей, идеи и интересы в одном месте.

> LinkA — создаём связи, а не просто аккаунты.

---

## ✨ Слоган

🌐 LinkA — там, где люди становятся связями

---

## 🧠 Почему название «LinkA»

Название LinkA происходит от слова link — «связь», «соединение».

В основе социальной сети лежит идея связей:
- 👤 между людьми  
- 💬 между сообщениями  
- 🧵 между интересами и сообществами  

Буква «A» в конце:
- делает название уникальным  
- легко читается и запоминается  
- символизирует Action / Access / Association

LinkA — это место, где каждая связь имеет значение.

## 🗄️ **Работа с базой данных**

🔸 При первом запуске:
 • если базы данных нет, она создаётся автоматически

🔸 При последующих запусках:
 • если база уже существует,
➜ используется старая база данных
➜ данные не удаляются

Это позволяет безопасно перезапускать приложение без потери информации.
